#Google MIC prototype

##This app only works correctly on Nexus 7 screen size(900px * 600px)

##Features
* TTS(Text to speech), STT(Speech to text) are supported
* Interactive UI/UX
* Navigation function
* Get current position button
* Autocomplete
* Finding nearby gas stations

###Used APIs
* Places API
* Directions API
* Distance Matrix API
